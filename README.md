# Dashboard Kendala Merchant EDC

Dashboard web modern untuk monitoring kendala harian merchant EDC menggunakan:

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Bootstrap 5
- Chart.js
- DataTables

## Menjalankan

1. Buka folder project di VS Code.
2. Jalankan dengan **Live Server**.
3. Dashboard akan otomatis mengambil data CSV dari Google Spreadsheet publik.

## Fitur

- KPI utama
- Filter dinamis
- Trend chart, top leader, top kendala, monitoring status
- Tabel detail dengan sorting, pagination, export Excel/CSV
- Auto refresh tiap 5 menit
- Dark mode toggle
- Loading state dan error handling
